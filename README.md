# 空塵計

## 命名規則

**資料庫欄位**=>全小寫蛇底式(ex:user_id)

**PHP類別**=>大駝峰(ex:UserController)

**PHP函數**=>第一個字小寫後面大駝峰(ex:userLogin())

**PHP變數**=>全小寫蛇底式(ex:$user_id)

**PHP常數**=>全大寫蛇底式(ex:$INPUT_MODE)

**SQL關鍵字**=>全大寫(ex:"SELECT * FROM `user` WHERE 1")

## Laravel 檔案命名規則

**Controller**=>大駝峰式命名法、複數

**Model**=>大駝峰式命名法、單數

**Blade**=>全小寫蛇底式
